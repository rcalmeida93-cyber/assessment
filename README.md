# BI Engineer Assessment

## 📌 Overview

This project presents a data modeling and analytics solution for a healthcare dataset.

---

## 🧠 Data Model

The data was structured using a star schema approach:

* Fact tables: appointments, transactions, events
* Dimension tables: users, providers, date

---

## 📊 Metrics

Key metrics:

* Total appointments
* Completion rate
* Total revenue
* Active users

---

## 💻 SQL Models

SQL logic includes:

* Data transformation
* KPI calculations
* Aggregations

---

## 📈 Dashboard

A dashboard mockup is included in the project.

## 📝 Documentation

## ✅ Conclusion

This solution supports operational and financial analysis.
